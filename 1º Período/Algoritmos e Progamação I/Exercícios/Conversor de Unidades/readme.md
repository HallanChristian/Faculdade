# Conversor de Unidades 

**Este programa em Java é um conversor de unidades simples.**

- No método main, cria um loop while que continuará executando até que o usuário escolha a opção "Sair" (opção 0).
- Dentro do loop, chama a função exibirMenu() para mostrar as opções disponíveis de conversão.
- Solicita ao usuário que escolha uma opção e armazena a escolha em uma variável escolha.
- Se o usuário escolher a opção 0, o programa exibe uma mensagem de encerramento e sai do loop.
- Caso contrário, chama a função realizarConversao(escolha) para executar a conversão escolhida.

**Aqui estão as funções exibirMenu() e realizarConversao(escolha):**

- exibirMenu(): Exibe as opções de conversão disponíveis para o usuário.
- realizarConversao(int escolha): Realiza a conversão de acordo com a opção escolhida pelo usuário.
- Para cada opção de conversão, o programa solicita ao usuário a quantidade a ser convertida e calcula o resultado usando as funções correspondentes. Em seguida, exibe o resultado da conversão.

### É um programa simples que permite ao usuário escolher entre converter diferentes unidades de medida e exibe o resultado da conversão.
